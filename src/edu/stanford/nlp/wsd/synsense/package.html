A package for learning joint models over the sense and syntax (argument) structure of individual verbs, and performing inference with them to label new instances of those verbs.

For more information, see Andrew, G., Grenager, T. and Manning, C. (2004). Verb sense and subcategorization: using joint inference to improve performance on complementary tasks. EMNLP 2004.

Contact Teg Grenager (grenager@stanford.edu) or Galen Andrew (pupochik@stanford.edu) for more information.
